        • public static java.nio.file.Path findFirstInZip(java.io.File zipFile, java.lang.String pattern)

          Retrouver dans un fichier zip le premier fichier qui répond au pattern.
          Cette méthode rend la recherche dans les fichier IN et OUT (de l'ATIH) plus facile.
          Exemple de recherche d'un fichier .tra.txt : 

           File zipOut = new File("C:/Local/GROUPAGE/2024/12/250127/INOUT/910019447.2024.0.20250129121840.out.zip");
 Path tra = FileUtils.findFirstInZip(zipOut, "*.tra.txt");
 if (tra != null) {
    Reader rdr = Files.newBufferedReader(tra);
    String content = FileUtils.toString(rdr);
    //...utilisation de "content"
  }
          throws:
          IOException Si erreur E/S
          Parameters:
          zipFile - Le fichier zip
          pattern - Le pattern à rechercher, de type "glob" ( cf. java.nio.file.FileSystem#getPathMatcher(java.lang.String) )
          Returns:
          L'objet java.nio.file.Path qui a été trouvé, ou null si rien n'a été trouvé

        • public static java.lang.String[] splitNameAndExtension(java.lang.String filename)

          Pour un nom de fichier, séparer le nom simple (ce qui est avant le dernier point) et l'extension (ce qui est après le dernier point). Exemple : 

           String[] parties = FileUtils.splitNameAndExtension("bonjour.xlsx");
          L'élément parties[0] contiendra "bonjour"
          L'élément parties[1] contiendra ".xlsx"
          Parameters:
          filename - Le nom de fichier
          Returns:
          Un tableau de deux éléments. L'élément 0 est le nom simple, et l'élément 1 est soit une chaîne vide s'il n'y avait pas de point dans le nom de fichier, soit le point + l'extension si un point était présent.

        • public static java.lang.String stripSuffix(java.lang.String name)

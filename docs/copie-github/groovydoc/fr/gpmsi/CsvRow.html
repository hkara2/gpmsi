Package: fr.gpmsi

[Java] Class CsvRow

    • fr.gpmsi.CsvRow 
    • public class CsvRow
extends java.lang.Object

      Encapsulation de CsvRow pour pouvoir utiliser les noms de colonne qui sont dans le ScriptStep de type csv. On peut utiliser en groovy des expressions telles que (s'il y a une colonne qui s'appelle codepostal) : row.codepostal et même changer la valeur par exemple row.codepostal = '91150'.

        • public boolean isEmpty()

          Retourner true si cette rangee est vide. En raison de l'utilisation de la classe CSVReader, on n'a pas accès à la ligne qui a été lue. On trouve donc que la ligne est vide si toutes les valeurs de la rangée sont vides. Ex (utilisation de Groovy) : 

             onItem {item->
       if (item.linenr == 1) return //ignorer l'en tete
       if (item.row.isEmpty()) return //ignorer lignes vides
       ...
          N.B. Lorsque Groovy est utilisé, item.row.empty est équivalent à item.row.isEmpty()
          Returns:
          True si la rangée est vide

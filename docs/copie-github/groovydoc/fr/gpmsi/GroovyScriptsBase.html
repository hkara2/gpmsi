Package: fr.gpmsi

[Java] Class GroovyScriptsBase

        • fr.gpmsi.GroovyScriptsBase 
        • public abstract class GroovyScriptsBase
extends groovy.lang.Script

          Classe de base des scripts Groovy du projet. Basé sur groovy.lang.Script, et rajoute les méthodes qui sont proposées à l'auteur de script : rsa, rss, etc. qui font de ces appels un mini-DSL (domain specific language). Maintenant après avoir enregistré chaque Closure, elle est exécutée directement plutôt qu'être juste stockée dans "steps". Ainsi le déroulement du script paraît plus naturel.

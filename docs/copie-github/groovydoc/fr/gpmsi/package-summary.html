Package fr.gpmsi

Package fr.gpmsi Description

Classes principales de Gpmsi, pour un fonctionnement avec des fichiers RSS, RSA, RHS, RHA, VIDHOSP, et des fichiers mono-niveau (FICHSUP, FICHCOMP, ...)
author:
hkaradimas

Package fr.gpmsi.pmsi_rules

Package fr.gpmsi.pmsi_rules Description

Mini-moteur de règles qui combine des classes de construction avec du code utilisateur en "closures"
  • Interface Summary 
    Interface Description
    PmsiCriterion Interface pour un critère PMSI qui sera évalué.
    PmsiRule Interface générale pour une "règle PMSI" minimaliste.
  • Class Summary 
    Class Description
    And Noeud "And" d'une AndOrExpr.
    AndOrExpr Expressions And et Or génériques.
    AoeNode Classe de base pour tous les noeuds impliqués dans une expression And-Or (AndOrExpr).
    ExampleNodeTreeWalker Exemple d'évaluation d'arbre ; ici prend "0" pour false, tout le reste pour true, et on évalue les opérations booléennes en parcourant un arbre à partir d'un noeud And, Or, Not ou Val.
    GenericPmsiCriterion Critère PMSI générique.
    GenericPmsiRule
    Not Noeud "Not" d'une AndOrExpr.
    Or Noeud "Or" d'une AndOrExpr.
    PmsiAllCriteria Ensemble de critères logiques PMSI (qui est lui-même un critère).
    PmsiAnd Identique à PmsiAllCriteria
    PmsiAnyCriterion Ensemble de critères logiques PMSI (qui est lui-même un critère).
    PmsiCriterionRule Classe de base de règle PMSI qui est juste construite sur un ou plusieurs critères PMSI, qui doivent tous être vérifiés pour que la règle soit déclenchée.
    PmsiNot Identique à PmsiNotCriterion
    PmsiNotCriterion Negation du sous-critère logique PMSI.
    PmsiOr Identique à PmsiAnyCriterion.
    PmsiRuleEngine Moteur de regles tres simpliste pour l'instant que l'on peut appeler pour évaluation de chaque RUM par exemple.
    Val Noeud "valeur" d'une AndOrExpr.

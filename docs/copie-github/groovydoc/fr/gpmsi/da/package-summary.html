Package fr.gpmsi.da Description

author:
hkaradimas Data Access Object (DAO) : classes pour faire un objet d'accès aux données, qui permet de faciliter l'accès à une table de base de données ou bien à des rangées d'un fichier .csv Réservé pour un usage futur.

        • abstract java.lang.Object stringToValue(java.lang.String str, DaPreferences prefs)

          Faire une valeur correctement typée, qui peut être insérée dans la base de données, à partir de la String passée. Les chaînes vides (longueur 0) sont converties en null.           throws:

          prefs - Les préférences

          L'objet du bon type

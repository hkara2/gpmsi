[Java] Class StringTableSelection

    • fr.gpmsi.StringTableSelection
    • All Implemented Interfaces and Traits:
      java.lang.Iterable
      public class StringTableSelection
extends java.lang.Object
implements java.lang.Iterable

      Objet de sélection qui désigne une sélection de rangées dans une StringTable. Ne pas utiliser cette sélection si la compsition des lignes de la StringTable a changé depuis la création de la sélection.

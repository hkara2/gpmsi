[Java] Class CsvDestination

    • fr.gpmsi.CsvDestination
    • All Implemented Interfaces and Traits:
      java.lang.AutoCloseable
      public class CsvDestination
extends java.lang.Object
implements java.lang.AutoCloseable

      Objet "destination" pour écrire dans un fichier ou un flux des valeurs qui seront séparées par une virgule ou un point-virgule. En interne, un objet com.opencsv.CSVWriter est utilisé pour écrire au format csv. Pour créer une destination, on utilise un des constructeurs disponibles. On écrit ensuite les valeurs une par une en utilisant la fonction f(Object) ou bien f(String). Une fois toutes les valeurs d'une rangée envoyées, on appelle la fonction endRow(). Lorsque l'on a fini d'envoyer toutes les rangées, on ferme la destination en appelant close(). L'interface java.lang.AutoCloseable est implémentée, donc on peut ouvrir cet objet dans un bloc "try-with-resources".
      La librairie opencsv utilisée est une ancienne version qui est plus simple pour le peu qui est utilisé ici. Il se peut qu'à l'avenir une nouvelle version sera utilisée. Si on veut d'autres valeurs pour les séparateurs, utiliser un CSVWriter directement.

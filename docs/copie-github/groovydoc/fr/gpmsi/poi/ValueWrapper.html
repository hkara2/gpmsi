Package: fr.gpmsi.poi

[Java] Class ValueWrapper

    • fr.gpmsi.poi.ValueWrapper 
    • public class ValueWrapper
extends java.lang.Object

      Encapsulation d'une valeur de cellule, pour pouvoir la transporter vers une autre cellule. Contient le format de date utilisé de manière à pouvoir le transporter avec la valeur et à ainsi le recopier également. Couvre 99% de mes besoins courants (ce n'est pas grave si les styles ne sont pas copiés). Pour l'instant j'ai encore des erreurs avec CellUtil#copyCell(Cell, Cell, org.apache.poi.ss.usermodel.CellCopyPolicy, org.apache.poi.ss.usermodel.CellCopyContext) qui est en bêta. Lorsque cela fonctionnera, il n'y aura plus besoin de cette classe.

                         Installation de gpmsi
            
L'installation est très simple :
Créer si cela n'existe pas encore le répertoire C:\app\gpmsi
Y dézipper le contenu de la distribution.

Par exemple pour la version v2.1 , on obtiendra 
un répertoire  C:\app\gpmsi\v2.1 .

Pour tester si la librairie fonctionne, exécuter dans une fenêtre
de commandes, la commande :

C:\app\gpmsi\v2.1\menu.bat

Cela devrait afficher le menu de gpmsi.

Ensuite aller dans le répertoire :

C:\app\gpmsi\v2.1\fichiers-ref

et double-cliquer sur le fichier batch :

installer-fichiers-ref.bat

Pour la documentation

Aller dans le répertoire C:\app\gpmsi\v2.1\docs , et double-cliquer sur index.html

Pour le développement :

Aller dans le répertoire "fichiers-ref" et exécuter le script "installer-fichiers-ref.bat".
Cela va créer un répertoire ".gpmsi" dans le répertoire de base de l'utilisateur.
Dans ce répertoire .gpmsi il va copier les deux répertoires ccam et cim.

Documentation pour Rss2Csv, dans pmsixml v. 1.6.1.


Options utilisables :

-in
    Donne le fichier d'entrée à transformer

-outdir
    Désigne le répertoire de sortie à utiliser. Par défaut c'est le répertoire courant.

-enc
    Désigne l'encodage à utiliser. Par défaut c'est l'encodage cp1252 sur Windows

-debug
    Active le mode débogage, qui écrit beaucoup d'informations sur la console.

-useidf
    Utilise le format de date international. Par défaut c'est le format de date francais qui est utilisé
    pour les dates dans les fichiers csv

-help
    Ecrit des informations d'aide sur la console

-prefix
    Désigne le préfixe qui sera utilisé dans chaque nom de fichier. Par défaut c'est la date du jour


Exemple d'utilisation de Rss2Csv :

cd "C:\Local\OPTIMISATION CODAGE\2017\CVMA\170320-lamda"

c:\app\pmsixml\1.6\scripts\pmsixml.bat Rss2Csv -in 910019447.2016.12.13032017145416.in\910019447.2016.12.rss.txt -prefix rss

Résultat :

5 fichiers générés :

rssDA.csv
rssDAD.csv
rssRUM.csv
rssSLI.csv
rssZA.csv
